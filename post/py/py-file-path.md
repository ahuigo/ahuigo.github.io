You cannot go past an anchor, or empty path:

    >>> p = PurePosixPath('/').parent
    PurePosixPath('/')
    >>> p = PurePosixPath('.').parent
    PurePosixPath('.')

Please use `Path.resolve()`:

    >>> PurePosixPath('foo/..').resolve()
