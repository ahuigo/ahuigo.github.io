# basic var
unix advanced program 2.8 提到基础系统数据类型：sys/types.h 

它们属于unix 系统类型：

    clock_t 时钟计数
    time_t  秒计数
    wchar_t 宽字符码
    dev_t
    fd_set
    mode_t 文件类型,创建模式
    pid_t   进程ID、进程组ID
    sigset_t    信号集
    size_t      无符号对象（如字符串）长度
    ssize_t     有符号长度
